Preamble

Website development was always something I knew I'd give a try oneday... and I guess that moment came now. So here you have it, a presentation of all I've learned regarding WebDev! Be it HTML, CSS, or JavaScript I hope I can learn a lot and show a lot here, as a documentation and presentation of my progress. I'll be the first to say that making this website was far more difficult then I imagined. The HTML and CSS in particular are quite different to the back-end coding I'm more familiar with. Regardless, I had fun building this site, and learned many new things on the journey, and with that I believe my mission was accomplished!
